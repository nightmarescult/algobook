# Что это?
Это хэндбук по соревновательному программированию. Моя цель — создать простой, понятный и подробный учебник по этой теме. Сайт работает на [mdbook](https://github.com/rust-lang/mdBook), невероятно быстром генераторе сайтов, написанном на Rust. Сайт вдохновлен [e-maxx](https://e-maxx.ru/algo), его английской альтернативой: [cp-algorihtms](https://cp-algorithms.com/index.html) и [algorithmica](https://ru.algorithmica.org/). К сожалению, в настоящее время он доступен только на русском языке (извините).

# Сборка
```sh
# вам нужен cargo
cargo install mdbook
git clone https://github.com/nightmarescult/algobook/ dest
cd dest
mdbook build
```
